# Java Application - Park

This is a simple Java application for the Park project, built using Gradle. The project is structured for testing with JUnit and includes Java 17 compatibility.

## Project Information

- **Group**: `uk.ac.sheffield.com1003.park`
- **Version**: `1.0-SNAPSHOT`
- **Main Class**: `uk.ac.sheffield.com1003.park.App`

## Requirements

- Java 17
- Gradle (used for build automation)
- JUnit 5 for testing

## Setup

1. Clone this repository to your local machine.
2. Make sure you have Java 17 installed.
3. Install Gradle if it is not already installed. You can follow the instructions here: https://gradle.org/install/
4. Navigate to the project directory and run:

## Commands
```bash
gradle build
```

```bash
gradle test
```

```bash
gradle run
```
